# AI Requirements Engineering Agent

An AI-powered tool that automatically converts source code into Software Requirements Specifications (SRS) using local LLMs (Ollama).

---

## Features

- Analyze source code files
- Generate Software Requirements (SRS)
- Extract:
  - Functional Requirements
  - Non-Functional Requirements
  - User Stories
  - Edge Cases
  - Assumptions
- CLI + Streamlit UI support
- Save generated requirements locally
- Fully local (no paid API required)

---

## Tech Stack

- Python
- Ollama (Llama 3.1)
- Streamlit
- Requests
